# README

## Installation

Clone [xled](https://github.com/scrool/xled.git) repository:
```bash
git clone https://github.com/scrool/xled.git
```

Install xled:
```bash
cd xled
pip install .
```

## Run

```bash
python3 twinkly-api.py
```
